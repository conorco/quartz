# Texte

| Action                    | Commande Latex            |
| ------------------------- | ------------------------- |
| mettre en gras            | `\bf{}`                   |
| mettre en italique        | `\emph{}`                 |
| souligner                 | `\underline{}`            |
| mettre en couleur (rouge) | `\textcolor{red}{texte}}` |
